# Getting started

```
git clone https://github.com/yashkal/.dotfiles.git
cd .dotfiles
make all
```

# Bash Tips

Run the following command to go to History Expansion in Bash man page

```
man -P 'less -p ^HISTORY\ EXPANSION' bash
```

Shows the table of contents for bash

```
man bash | grep '^[A-Z]'
```
